# hikikomori

## Build
    $ make
    
## Run
    $ ./hikikomori %THREADS_NUM%

Example:

    $ ./hikikomori 500
    
If you have a decent server (not a VPS), you probably want to adjust your limits.conf and try 2-3k threads.
